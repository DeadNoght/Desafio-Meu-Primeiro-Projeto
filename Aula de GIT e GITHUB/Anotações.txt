Para acessar o GITHUB: github.com


Minhas considerações: O intuito é fazer com que pessoas de locais diferentes possam se ajudar fazendo um 'pushing' dos códigos e ir melhorando e desenvolvendo o código.
É importante usar o comando 'git clone' + link SSH ou HTTPS no git bash para fazer o upload do código para a pasta que você quiser.
